Katalog open-source nástrojů

Websto je jednoduchý přehled open-source aplikací zaměřených na bezpečnost, soukromí a technickou kontrolu nad vlastními daty.

Vyber nástroj, klikni a zjisti, jak ho používat správně a bezpečně.

Websto používá pouze veřejně dostupné informace. Web neukládá žádná osobní data a nepoužívá cookies.

Projekt je hostován na GitHub Pages.
